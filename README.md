# 🎓 Student Result Manager (Python CLI)

A simple command-line based student management system built using Python.

## 📌 Features

* Add student details (ID, Name, Marks)
* View all students in tabular format
* Check pass/fail result
* Data stored permanently using JSON

## 🛠️ Tech Used

* Python
* JSON (for storage)

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/AADITYAKUMBARE215/STUDENT-RESULT-MANAGER.git
   ```

2. Navigate to project folder:

   ```bash
   cd "STUDENT RESULT MANAGER"
   ```

3. Run the program:

   ```bash
   python main.py
   ```

## 📂 File Structure

* `main.py` → Main program
* `students.json` → Stores student data

## 💡 About

This project is a beginner-friendly CLI application to understand:

* File handling in Python
* JSON data storage
* Dictionaries and loops
* Basic user interaction
